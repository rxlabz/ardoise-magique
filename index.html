Ardoise magique

Reproduisez l'image ci-dessus à l'aide de ces 4 commandes :

  • trait( departX, departY, arriveeX, arriveeY, couleurOptionnelle* )
  • rect( positionX, positionY, largeur, hauteur, couleurOptionnelle* )
  • rond( centreX, centreY, rayon, couleurOptionnelle* )
  • efface()

Exemple : trait( 50, 50, 150, 150, 'r')

* Couleurs disponibles :
  • "r" : rouge
  • "v" : vert
  • "b" : bleu
  • "o" : orange
( Par défaut, les traits sont dessinés en noir )
